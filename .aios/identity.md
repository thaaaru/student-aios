# AIOS Identity

Philosophy: Use AI to learn faster, think better, and build useful things, not to avoid learning.

## Checklist
- Student: name, university, year, main interest
- Current course: course name, main topics, tools, deadlines
- Learning goals: understand topics, build projects, ask better AI questions
- Strengths: curiosity, practice, asking questions
- Weaknesses: rushing, copying, unclear planning
- Learning style: simple English, steps, small examples, real-world examples
